VR ¬∑ LLM

LLM-Assisted VR Training Framework

A VR framework that uses LLaMA-based models to generate task instructions, answer user questions, and monitor progress inside a virtual cleanroom and manufacturing environment.

  • Integrates LLaMA 3.1, Whisper, and eye tracking in Unreal Engine.
  • Structured JSON outputs for actions, steps, and component-level monitoring.
  • Hosted on Hugging Face with real-time latency analysis for VR deployment.

Avatar ¬∑ Ergonomics

Dynamic Avatar for Ergonomic Guidance(On-Going project)

Real-time avatar system that demonstrates tasks in VR while embedding ergonomic principles to reduce risk and improve posture during training.

  • Inverse kinematics and body calibration using motion capture data.
  • Avatar actions aligned with ergonomic standards such as ISO 11226.
  • Designed for future integration with user-specific ergonomic scoring.
